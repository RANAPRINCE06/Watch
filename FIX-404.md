# FIX VERCEL 404 ERROR

## Quick Fix (2 options):

### Option 1: Change Root Directory in Vercel
1. Go to your Vercel project dashboard
2. Click "Settings" → "General"
3. Find "Root Directory"
4. Change to: `client`
5. Click "Save"
6. Go to "Deployments" → Click "..." → "Redeploy"

### Option 2: Redeploy with Correct Settings
1. Delete current deployment in Vercel
2. Click "Add New" → "Project"
3. Import your repo again
4. **IMPORTANT**: Set Root Directory to `client`
5. Framework: Next.js (auto-detected)
6. Add environment variables:
   ```
   NEXT_PUBLIC_API_URL=https://your-backend-url.onrender.com/api
   NEXT_PUBLIC_RAZORPAY_KEY_ID=rzp_test_xxxx
   NEXT_PUBLIC_STRIPE_KEY=pk_test_xxxx
   ```
7. Deploy

## Then Push Updated Files:
```bash
git add .
git commit -m "Fix Vercel 404 - add config files"
git push origin main
```

Your site should work after redeploying!
